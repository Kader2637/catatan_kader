Konsep Dasar

Anatomi Sintaks PHP

01 Tag Pembuka & Penutup

Setiap kode PHP harus berada di dalam tag <?php ... ?>. Jika file hanya berisi kode PHP tanpa HTML, praktik terbaik adalah tidak menggunakan tag penutup ?> untuk menghindari error spasi putih (whitespace). 

<?php
// Ini adalah komentar satu baris
/* Ini adalah komentar
   multi-baris 
*/

echo "Halo, Dunia!";

02 Variabel & Deklarasi

Di PHP, variabel dimulai dengan tanda dolar ($) dan bersifat Loosely Typed (tidak perlu mendefinisikan tipe data di awal).

<?php
$namaLengkap = "Abdul Kader"; // String
$umur = 24;                  // Integer
$ipk = 3.85;                 // Float / Double
$statusAktif = true;         // Boolean

// Menggabungkan string (Concatenation) menggunakan titik (.)
echo "Nama saya " . $namaLengkap . ", umur saya " . $umur . " tahun.";
Kader Note (Penamaan Variabel):

Variabel PHP bersifat Case-Sensitive. $Nama dan $nama adalah dua variabel yang berbeda. Gunakan konvensi camelCase (seperti `$namaLengkap`) atau snake_case (seperti `$nama_lengkap`) secara konsisten.

03 Tipe Data Kompleks & Debugging

Selain tipe data dasar (skalar), PHP memiliki tipe data gabungan yaitu Array dan Object. Untuk mengecek struktur tipe data ini (debugging), kita wajib tau fungsi var_dump() atau print_r().

<?php
// Array Index biasa
$hobi = ["Coding", "Membaca", "Gaming"];

// Array Asosiatif (Key => Value)
$mahasiswa = [
    "nim" => "1905001",
    "nama" => "Abdul Kader",
    "jurusan" => "Teknik Informatika"
];

// Menampilkan isi array (Debugging)
var_dump($mahasiswa); 
// print_r($hobi); juga sering dipakai

04 Operator Penting

Dalam PHP kita sering berinteraksi dengan tipe data menggunakan operator, beberapa yang paling sering digunakan adalah:

  • Aritmatika: Penjumlahan (+), Pengurangan (-), Perkalian (*), Pembagian (/), Modulus (%).
  • Assignment: =, +=, -=.
  • Perbandingan: Sama dengan (==), Identik tipe data (===), Tidak sama dengan (!=), Lebih besar/kecil (>, <).
  • Logika: AND (&&), OR (||), NOT (!).
