Modul 2: Struktur Kendali (Control Flow)

Modul 02

"Mengendalikan Alur Logika Aplikasi"

Tanpa struktur kendali, program hanya akan berjalan lurus dari baris 1 sampai akhir. Dengan struktur kendali, program bisa "berpikir", mengambil keputusan, dan mengulang tugas (otomatisasi).

Pengkondisian (Kondisional)

01 IF, ELSE IF, dan ELSE

Mengevaluasi kondisi bernilai boolean (True/False). Jika True, blok kode dijalankan.

<?php
$nilaiUjian = 85;

if ($nilaiUjian >= 90) {
    echo "Grade: A";
} elseif ($nilaiUjian >= 80) {
    echo "Grade: B"; // Baris ini yang akan dicetak
} else {
    echo "Belajar lebih giat lagi!";
}

02 Switch Case & Match (PHP 8.0+)

Gunakan ini jika Anda membandingkan satu variabel dengan banyak nilai pasti. match adalah versi modern dan lebih aman dari switch. 

<?php
$role = "admin";

// Menggunakan Switch
switch ($role) {
    case "admin":
        echo "Selamat datang Administrator";
        break;
    case "user":
        echo "Selamat datang Pengguna";
        break;
    default:
        echo "Akses Ditolak";
}

// Menggunakan Match (Lebih ringkas, return value langsung)
$pesan = match ($role) {
    'admin' => 'Selamat datang Administrator',
    'user' => 'Selamat datang Pengguna',
    default => 'Akses Ditolak',
};
echo $pesan;

Perulangan (Looping)

Perulangan digunakan saat kita ingin menampilkan data dari Database ke dalam tabel HTML berkali-kali tanpa menulis kode berulang.

For Loop & Foreach Loop

<?php
// For Loop (Umumnya untuk iterasi dengan batasan angka)
for ($i = 1; $i <= 5; $i++) {
    echo "Data ke-" . $i . "<br>";
}

// Foreach Loop (SANGAT SERING DIGUNAKAN UNTUK ARRAY & DATABASE)
$kaderList = ["Budi", "Andi", "Siti"];

foreach ($kaderList as $kader) {
    echo "Nama: " . $kader . "<br>";
}
